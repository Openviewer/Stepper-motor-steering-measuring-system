# Stepper-motor-steering-measuring-system

## Objective
This project aims to measure the steering condition (i.e., typically two directions) of stepper motor on the 3D printer. 

## Test stage 
### Stage 1
#### Setup
- Raspberry Pi 3 B+
- 5V Stepper Motor
- Jumper wire pack
- Breadboard
- ULN2003
- Power supply
